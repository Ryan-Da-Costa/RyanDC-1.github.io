Abstract

This application can be used to mimic human conversation. Chatbots can interact with humans and answer basic queries.


Working

The application is implemented using flutter and dialogflow API. The API consists of Intents which is used to match the queries of the user and respond according to the query. The application communicates with the API and send the text data that had been entered by the user, The API than returns a response to the query by matching it to an intent.


Technologies used

Flutter, Dialogflow API.
