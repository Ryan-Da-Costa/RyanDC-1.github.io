Abstract

This device will help a visually impaired person to navigate in the surrounding environment using object recognition, obstacle detection and face recognition. The device will communicate with the user using beep sounds or vibrational motion and speech output. It can also detect and recognise people


Working

The device consists of three modules:
1) Face recognition module
2) Object recognition module
3) Obstacle detection module
The face recognition module uses HOG-SVM algorithm to detect a persons face and is started by the object recognition algorithm once a person is detected. The deice outputs the name of the person using a speech output. device also allows the user to store new faces by pressing a button which captures a set of images of the contacts face and stores it in memory. The stored images are than trained using the algorithm. The face recognition module can help the user to identify and interact with people.
The object recognition module uses YOLO algorithm to detect and recognise objects. Once the device recognises an object present in the users path, it outputs the name of the object as a speech output. The user will be aware of all the object in their surrounding and will be able to navigate around them and interact with them.
The obstacle detection module is implemented using an ultrasonic sensor and an ESP-32 microcontroller. The ultrasonic sensors are used to detect any obstacles that may be present in the users path. Once the threshold distance is reached, it will output a beep sound/vibration indicating that there is an obstacle.

The algorithms are implemented on a raspberry Pi. A camera connected to the raspberry Pi will deliver the input feed for the algorithm. The device will consist of a wearable device along with a walking stick. The wearable device will house the camera which will be used for face and object recognition and the walking stick will house the ultrasonic sensors for obstacle detection.
