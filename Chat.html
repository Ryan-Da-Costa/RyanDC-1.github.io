Abstract

This is a chat application that can be used to communicate with other users. Firebase provides real-time database service. The chat application communicates with firebase API and obtains all the messages present in the firebase collection. Upon composing a message we can add messages to a collection which can be received by other users.


Working

Firebase authenticates the user upon login. When a user sends a message, the text, users name and the timestamp is sent to the firebase collection. The receiver will receive the messages in order of their timestamp.


Technologies used

Flutter, Firebase API.
